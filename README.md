# 时光邮局（Time Post Office）

> 一个温暖的回忆珍藏站，记录生活中的美好瞬间，与 AI 小邮差一起守护时光。

## 🏠 在线访问

**Vercel 部署：** https://time-post-office-blue.vercel.app/

## ✨ 核心功能

### 回忆珍藏
- 📸 **上传照片**：将珍贵回忆上传并保存到本地浏览器
- 🏷️ **AI 标签**：自动为回忆生成标签（旅行、家人、美食等），支持手动增删
- 🔍 **搜索筛选**：按关键词搜索回忆，按时间、心情、标签筛选浏览
- ☁️ **关键词云**：可视化展示所有回忆标签，大小不一如云朵般散布

### 心情日历
- 📅 **打卡日历**：每日记录心情，生成温暖的心情日历
- 🔥 **连续打卡**：统计连续记录天数，可视化 streak 展示
- 📊 **数据统计**：心情分布、上传数量、打卡天数一目了然

### AI 小邮差
- 🤖 **AI 对话**：与"小邮差"聊天，分享回忆背后的故事
- 🎙️ **语音输入**（Microsoft Edge 浏览器支持）：语音与小邮差对话
- 🔊 **朗读功能**：选择喜欢的声音朗读回忆文字

### 时光盲盒
- 🎁 **每周盲盒**：每周可开启一次，随机抽取一段回忆唤醒
- 🧩 **拼图解锁**：解锁拼图后可唤醒隐藏回忆，与 AI 对话记录

### 分享与记录
- 🎑 **明信片生成**：将回忆生成精美明信片，支持多种字体风格
- 📈 **时光年报**：AI 自动生成年度/月度回忆报告，用小邮差的口吻讲述温暖故事
- 💊 **时光胶囊**：写给未来的信，定时解锁

### 背景音乐
- 🎵 **按心情播放**：不同心情对应不同风格音乐（爵士、钢琴、八音盒、Lo-Fi 等）
- 🔄 **自动循环**：播放完一首自动播放下一首，循环播放

### 数据管理
- 💾 **导出/导入**：支持导出完整数据备份，导入时智能合并
- ⏰ **自动备份提醒**：照片超过 30 张或 7 天未备份时友好提醒
- 📢 **回忆推送**：自动推送珍贵回忆，支持自定义推送时间阈值

## 🛠️ 技术栈

- **前端**：HTML5、CSS3、JavaScript（ES6+）
- **样式**：Tailwind CSS + 自定义 CSS 变量
- **图标**：Lucide Icons
- **部署**：Vercel
- **数据存储**：浏览器 localStorage（本地存储）

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/alibabahappyman/TimePostOffice.git

# 进入项目目录
cd TimePostOffice

# 本地预览（使用任意静态服务器）
# 方式1：Python
python -m http.server 3456

# 方式2：Node.js
npx serve .

# 方式3：VS Code Live Server 插件
# 在 VS Code 中右键 index.html → "Open with Live Server"
```

然后打开浏览器访问 `http://localhost:3456`。

## ⚠️ 重要提示

### 数据存储说明
本项目使用 **浏览器 localStorage** 存储数据，所有数据保存在本地浏览器中：

- ✅ 数据完全存储在您的设备上，无需联网即可使用
- ⚠️ **清除浏览器缓存/使用无痕模式**会导致数据丢失
- ⚠️ **更换浏览器或设备**数据不会同步

### 建议备份策略
1. **定期导出**：点击右侧浮动按钮 → 回忆推送设置 → 数据管理 → 导出数据
2. **妥善保存**：将导出的 JSON 文件保存在安全位置（如云端硬盘、U盘）
3. **导入恢复**：需要恢复时，通过导入功能将数据还原

### 浏览器兼容性
- **推荐**：Microsoft Edge（支持语音输入功能）
- **支持**：Chrome、Firefox、Safari（最新版本）
- **语音输入**：仅 Microsoft Edge 浏览器支持（使用微软语音服务器，国内网络可访问）
